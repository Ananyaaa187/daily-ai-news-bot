\# Daily AI News Telegram Bot



This project uses n8n to send daily tech news to Telegram.



\## Features

\- Runs automatically at 9:00 AM

\- Fetches news from NewsAPI

\- Sends updates via Telegram bot



\## Setup

1\. Import workflow.json into n8n

2\. Add your Telegram bot token

3\. Replace YOUR\_API\_KEY with your NewsAPI key
